# RBAC 权限管理系统前端

本项目前端使用 vue 和 element-ui 开发，前端路由、按钮权限控制、操作数据加密传输。

配套 node 后端：https://github.com/xh-shen/egg-rbac-server.git。

## 安装依赖

```bash
$ yarn
```

## 启动项目

```bash
$ yarn run serve
```

如果启动了配套后端， 超级管理员账号：admin 密码：111111

## 打包项目

```
yarn run build
```
